# Index

Master catalog of all vault contents. Tables are generated by Dataview and update automatically.

---

## Sources: Unread / Unprocessed

```dataview
TABLE url, medium, saved
FROM "sources"
WHERE status = "unread" OR status = "unprocessed"
SORT saved ASC
```

## Sources: Read but Not Processed

```dataview
TABLE url, medium, saved
FROM "sources"
WHERE status = "read"
SORT saved ASC
```

## All Sources (by medium)

```dataview
TABLE url, saved, status
FROM "sources"
SORT medium ASC, saved DESC
```

---

## Atoms

```dataview
TABLE aliases, confidence, tags
FROM "atoms"
SORT confidence DESC, file.name ASC
```

### Orphan Atoms (no cites, no backlinks)

```dataview
LIST
FROM "atoms"
WHERE !cites AND length(file.inlinks) = 0
```

---

## Glossary

```dataview
TABLE term, domain
FROM "glossary"
SORT domain ASC, term ASC
```

### Unclaimed Glossary Terms (no backlinks)

```dataview
LIST
FROM "glossary"
WHERE length(file.inlinks) = 0
```

---

## Topics

### Concept Maps

```dataview
TABLE tags, file.ctime
FROM "topics/concepts"
SORT file.name ASC
```

### Active Projects

```dataview
TABLE status, tags
FROM "topics/projects"
WHERE status = "active"
SORT file.ctime DESC
```

### Open Research Questions

```dataview
TABLE question, tags
FROM "topics/research"
SORT file.ctime DESC
```

---

## Maintenance Flags

### Sources older than 30 days still unread

```dataview
LIST url
FROM "sources"
WHERE status = "unread" AND date(saved) < date(today) - dur(30 days)
SORT saved ASC
```

### Atoms with no part-of link (uncategorized)

```dataview
LIST
FROM "atoms"
WHERE !part-of
```
